# Lumière | Premium Beauty Salon Website

Lumière is a premium, high-end landing page for a luxury beauty salon offering expert hair styling, makeup, and nail services. The project is crafted with visual excellence at its core, combining an elegant modern aesthetic with responsive layouts, micro-animations, and an interactive reservation form.

---

## ✨ Features

- **Luxury Design & Aesthetics:** Beautiful styling combining the classic elegance of the `Playfair Display` typeface with the modern clarity of the `Inter` font.
- **Service Catalog:** Detailed showcases for **Hair Artistry**, **Nail Care**, and **Makeup & Bridal** complete with custom overlay interactions.
- **Fully Responsive & Fluid:** Adapts seamlessly across all devices, from high-resolution desktop screens to smartphones, including an interactive mobile navigation drawer.
- **Interactive Booking Form:** A functional, client-side validated appointment booking form with custom inputs for selecting date, time, and service type.
- **Client Stories:** Interactive testimonial section featuring real customer experiences.
- **Newsletter Subscription:** Interactive footer sign-up form for promotions and updates.

---

## 🛠️ Technology Stack

- **Markup:** Semantic HTML5
- **Styling:** Custom Vanilla CSS3 (utilizing CSS custom properties, grid layouts, flexbox, and performance-optimized transition keyframes)
- **Scripting:** Modern Vanilla JavaScript (dynamic navbar controls, active scroll states, mobile menu toggle, and form validation)
- **Icons:** FontAwesome v6.4

## 🚀 Getting Started

To launch and explore the project locally on your machine:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Divinaann/beauty-salon.git
   cd beauty-salon
   ```
2. **Launch the Website:**
   - Double-click the `index.html` file to open it in your browser.
   - Alternatively, use a development server extension like **Live Server** in VS Code to run it at a local address (e.g., `http://127.0.0.1:5500`).
